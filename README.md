## Project Overview
A fully functional women's fashion e-commerce store built on 
WordPress for a Pakistani clothing brand.

## Tech Stack
- **Theme:** Hello Elementor
- **Page Builder:** Elementor Pro
- **E-Commerce:** WooCommerce
- **SEO:** Rank Math SEO
- **Speed:** WP Rocket (Caching & Optimization)
- **Tracking:** Facebook Pixel

## What I Built
- Custom homepage design using Elementor
- WooCommerce product catalog setup
- Mobile responsive design
- On-page SEO configuration (Rank Math)
- Website speed optimization (WP Rocket)
- Payment gateway integration
- Facebook Pixel & event tracking setup
